Gerenciador de Tarefas e Usuários (Console)

Este é um projeto que desenvolvi para aplicar o que aprendi sobre Java. O foco principal foi organizar o código de um jeito profissional, separando a lógica de negócios da parte que salva os dados (Padrão DAO).

O que eu fiz aqui:
- No projeto de **Tarefas**, usei Java Streams para criar os filtros de prioridade e status, o que deixou o código muito mais limpo.
- No projeto de **Usuários**, trabalhei com validações de CPF e e-mail para garantir que o sistema não aceite dados errados.
- Configurei o **Maven** do zero para gerenciar o projeto e garantir que ele rode corretamente com o Java mais novo.

A estrutura está dividida em camadas (Model, DAO e Service) para facilitar a manutenção no futuro.
